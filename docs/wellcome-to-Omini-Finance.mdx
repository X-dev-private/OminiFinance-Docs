A DAO OminiFinance prioriza a integração das economias das maiores blockchains, promovendo a interoperabilidade entre plataformas como Bitcoin, Ethereum, Binance Smart Chain e Solana. Acreditamos que a colaboração entre essas redes pode criar um ecossistema financeiro mais eficiente, ágil e acessível, impulsionando inovações e novos modelos de negócios.

Ao unir forças, buscamos oferecer maior segurança, escalabilidade e oportunidades de diversificação para nossos membros, enquanto trabalhamos no desenvolvimento de soluções que facilitam a troca de ativos e dados entre as blockchains. A OminiFinance está comprometida em liderar essa transformação.

Como fazer um fork do repositório

Caso queira contribuir com o projeto, siga os passos abaixo para fazer um fork:

Acesse o repositório no GitHub: 
```
https://github.com/X-dev-private/ominiFinance
```
No canto superior direito da página do repositório, clique no botão Fork.

Isso criará uma cópia do repositório no seu próprio GitHub.

Clone o repositório forkado para a sua máquina:
```
git clone https://github.com/X-dev-private/ominiFinance.git
```
Faça as alterações desejadas, commit e envie para o seu repositório forkado.

Para contribuir com o projeto original, abra um Pull Request no repositório principal.

Como instalar o projeto

Para instalar as dependências do projeto, siga os passos abaixo:

Certifique-se de ter o Node.js instalado em sua máquina.

Clone o repositório do projeto:
```
git clone https://github.com/X-dev-private/ominiFinance.git
```
Acesse a pasta do projeto:
```
cd ominiFinance
```
Instale as dependências com:
```
npm install
```
Como rodar o projeto

Após instalar as dependências, inicie o servidor de desenvolvimento com:
```
npm run dev
```
Isso iniciará o ambiente de desenvolvimento e o projeto estará acessível no navegador.
